# [CVPR 2025] SURGEON: Memory-Adaptive Fully Test-Time Adaptation via Dynamic Activation Sparsity
This is the official repository for SURGEON, a novel method designed to reduce memory cost in fully test-time adaptation while maintaining competitive accuracy improvements. By introducing a dynamic activation sparsity strategy, SURGEON prunes activations at layer-specific and adaptive ratios in a data-sensitive manner, achieving a state-of-the-art (SOTA) trade-off between accuracy and memory efficiency.
